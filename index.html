Selected work

三个能被面试官快速追问下去的项目故事。

Problem

AI 生成字幕功能有真实使用需求，但用户付费意愿和商业化路径不清晰，需要从漏斗里找到能被实验验证的杠杆。

Action

拆解付费瓶颈，设计 A/B 实验，围绕权益感知、转化触点和投诉反馈持续迭代，推动策略上线。

Result

GMV 每天增长数百万，付费率提升超过 1 个百分点，投诉率降低超 70%。另推进音色克隆功能从 0 到 1，完整经历 PRD、协作、上线和数据监控。

Problem

教育场景下有大量试卷和报告自动化处理需求，模型效果、成本控制、稳定交付都需要被产品化管理。

Action

识别流程瓶颈，对比 LLM 技术选型，搭建成本评估体系和数据看板，让交付效果可以持续追踪。

Result

支持 10 万+ 份报告稳定交付，识别成功率显著提升，并沉淀出后续项目可复用的数据基建。

Problem

日常活动规划依赖人工，效率低且个性化不足，需要把用户偏好、约束条件和生成策略组织成可靠流程。

Action

设计状态机 + LLM 混合驱动架构，用 Vibe Coding 快速实现，并建立基础评测体系验证输出质量。

Result

验证了独立构建 AI 产品的完整链路能力，覆盖架构决策、快速实现、效果评估和产品表达。
